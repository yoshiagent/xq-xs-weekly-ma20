選股腳本（Screener）· 週K線

週MA20 量縮回測選股

篩選週MA20 上揚、曾有正乖離過大、目前量縮修正回均線附近的個股

策略邏輯

條件一
週MA20 斜率向上
本週均線值 > 上週均線值，確認中期多頭趨勢仍在
Average(Close,20) > Average(Close,20)[1]
條件二
近期曾有顯著正乖離
回看期間（預設 13 週）內，收盤最高點曾高於週MA20 達門檻幅度，代表股價曾強勢衝高過熱
Highest(Close, 13) / MA20 − 1 ≥ 8%
條件三
量縮修正（非恐慌賣壓）
本週成交量 < 10週均量的 85%，確認回調是自然換手而非主力出貨
Volume / Average(Volume,10) < 0.85
條件四
乖離收斂至均線附近
當前收盤與週MA20的乖離率絕對值 ≤ 3%，代表修正已完成，均線支撐帶到位
|（Close − MA20）/ MA20| ≤ 3%
✅ 觸發輸出
四條件同時成立 → ret = 1
股票進入選股結果，同步輸出五個九宮格欄位供分析
使用建議：篩出結果後，建議搭配週K線圖人工確認，重點觀察下影線支撐與量縮是否明顯；等下週週K收線確認守穩後再評估進場。

可調整參數一覽

變數名稱 預設值 說明 調整建議
paramDevHigh 8.0 % 回看期間內，股價曾高於週MA20 多少才算「乖離過大」 篩出太多 → 調高至 10–12% 篩出太少 → 降至 5–6%
paramDevNear 3.0 % 當前乖離率絕對值需收斂至此範圍內，才視為回到均線附近（±N%） 更貼近均線 → 調至 2% 容許範圍寬一點 → 調至 4%
paramVolRatio 0.85 量縮閾值，本週量 / 均量須低於此比例才符合量縮條件 要求更明顯量縮 → 調至 0.70 放寬量縮標準 → 調至 0.95
paramLookback 13 週 往前看幾根週K棒確認曾有高乖離（13週≒3個月） 抓更長週期高點 → 調至 20–26 週 只看近期 → 調至 8 週
paramVolMa 10 週 計算均量所用的週期數，作為量縮比較基準 一般不需調整，維持 10 週即可

九宮格輸出欄位

OutputField 1
當前乖離率（%）
OutputField 2
期間最大正乖離（%）
OutputField 3
量 / 均量比值
OutputField 4
週MA20 數值
OutputField 5
收盤價

XScript 完整程式碼

週MA20量縮回測選股.xs 
// ============================================================
// 策略名稱：週MA20量縮回測選股
// 執行環境：選股腳本（Screener）/ 週K線圖
// 策略邏輯：
//   1. 週MA20 斜率向上（多頭趨勢確認）
//   2. 近期曾有顯著正乖離（股價一度強勢衝高）
//   3. 目前以量縮方式回測（修正非恐慌賣壓）
//   4. 當前乖離率已收斂至週MA20附近
// 版本日期：2026-05-08
// 修正紀錄：
//   v2 - 使用 GetField("Close","W") 明確取週資料
//   v3 - 改用 xf_GetValue("W",Wma20,1) 取前一週MA20，消除警告
// ============================================================

// 確保足夠的歷史 K 棒長度（週K 20MA + 回看 26 週）
SetTotalBar(260);

// ── 參數設定 ────────────────────────────────────────────
// 正乖離觸發門檻（%）：股價曾高於週MA20 多少才算「乖離過大」
var: paramDevHigh(8.0);

// 回測完成門檻（%）：當前乖離率的絕對值小於此值，視為回到均線附近
var: paramDevNear(3.0);

// 量縮判斷：目前量 < 均量的比例倍數（<1 = 量縮）
var: paramVolRatio(0.85);

// 回看週數：往前幾根週K棒內確認曾有高乖離
var: paramLookback(13);

// 均量計算期數（週）
var: paramVolMa(10);

// ── 週資料（明確指定頻率）───────────────────────────────
var: wClose(0), wVol(0), Wma20(0);

wClose = GetField("Close", "W");                            // 週收盤價
wVol   = GetField("Volume", "W");                           // 週成交量
Wma20  = Average(GetField("Close", "W"), 20);               // 週MA20

// ── 計算核心變數 ─────────────────────────────────────────
var: devNow(0);
var: highestClose(0), highestDev(0);
var: volMaVal(0), volRatioNow(0);

// 當前日收盤對週MA20 的乖離率（%）
// 用 GetField("Close","D") 取日收盤，與週MA20比較
if Wma20 <> 0 then
    devNow = (GetField("Close", "D") - Wma20) / Wma20 * 100
else
    devNow = 0;

// 回看期間內的最高週收盤（用來計算歷史最大正乖離）
highestClose = Highest(wClose, paramLookback);

// 歷史最大正乖離率（%）
if Wma20 <> 0 then
    highestDev = (highestClose - Wma20) / Wma20 * 100
else
    highestDev = 0;

// 週均量
volMaVal = Average(wVol, paramVolMa);

// 當前量與均量的比值
if volMaVal <> 0 then
    volRatioNow = wVol / volMaVal
else
    volRatioNow = 1;

// ── 布林輔助旗標 ─────────────────────────────────────────
var: isMaRising(false);     // 週MA20 斜率向上
var: hadHighDev(false);     // 曾有顯著正乖離
var: isNearMa(false);       // 當前已回到均線附近（乖離收斂）
var: isVolShrink(false);    // 量縮確認

// 條件一：週MA20 上揚
// xf_GetValue("W",Wma20,1) = 取前一根週K棒的 Wma20（官方正確寫法）
if Wma20 > xf_GetValue("W", Wma20, 1) then
    isMaRising = true
else
    isMaRising = false;

// 條件二：回看期間內曾有正乖離 >= paramDevHigh
if highestDev >= paramDevHigh then
    hadHighDev = true
else
    hadHighDev = false;

// 條件三：當前乖離率絕對值 <= paramDevNear（回到均線附近）
// 允許略低於MA20（-paramDevNear ~ +paramDevNear）
if devNow <= paramDevNear and devNow >= (-1 * paramDevNear) then
    isNearMa = true
else
    isNearMa = false;

// 條件四：量縮（本週成交量 < 均量 * paramVolRatio）
if volRatioNow < paramVolRatio then
    isVolShrink = true
else
    isVolShrink = false;

// ── 選股觸發 ─────────────────────────────────────────────
if isMaRising = true
    and hadHighDev = true
    and isNearMa = true
    and isVolShrink = true
then begin
    ret = 1;

    // 九宮格輸出欄位
    OutputField1(devNow,                          "當前乖離率(%)");
    OutputField2(highestDev,                      "期間最大正乖離(%)");
    OutputField3(volRatioNow,                     "量/均量比值");
    OutputField4(Wma20,                           "週MA20");
    OutputField5(GetField("Close", "D"),          "日收盤價");
end;
