週MA20 量縮回測選股

篩選週MA20 上揚、曾有正乖離過大、目前量縮修正回均線附近的個股

策略邏輯

條件一
週MA20 斜率向上
本週均線值 > 上週均線值，確認中期多頭趨勢仍在
Wma20 > xf_GetValue("W", Wma20, 1)
條件二
近期曾有顯著正乖離
回看期間（預設 13 週）內，週收盤最高點曾高於週MA20 達門檻幅度，代表股價曾強勢衝高過熱
Highest(wClose, 13) / Wma20 − 1 ≥ 8%
條件三
峰值已在 2 週前以上
最高收盤距今至少 2 根週K棒，確認回測已啟動中，排除本週才見頂的情形
NthHighestBar(1, wClose, 13) ≥ 2
條件四
乖離收斂至均線附近
當前日收盤與週MA20的乖離率絕對值 ≤ 3%，代表修正已完成，均線支撐帶到位
|（Close_D − Wma20）/ Wma20| ≤ 3%
條件五
量縮修正（非恐慌賣壓）
本週成交量 < 10週均量的 85%，確認回調是自然換手而非主力出貨
wVol / Average(wVol, 10) < 0.85
✅ 觸發輸出
五條件同時成立 → ret = 1
股票進入選股結果，同步輸出九個九宮格欄位供分析
使用建議：篩出結果後，建議搭配週K線圖人工確認，重點觀察下影線支撐與量縮是否明顯；等下週週K收線確認守穩後再評估進場。

可調整參數一覽

變數名稱 預設值 說明 調整建議
paramDevHigh 8.0 % 回看期間內，股價曾高於週MA20 多少才算「乖離過大」 篩出太多 → 調高至 10–12% 篩出太少 → 降至 5–6%
paramDevNear 3.0 % 當前乖離率絕對值需收斂至此範圍內，才視為回到均線附近（±N%） 更貼近均線 → 調至 2% 容許範圍寬一點 → 調至 4%
paramVolRatio 0.85 量縮閾值，本週量 / 均量須低於此比例才符合量縮條件 要求更明顯量縮 → 調至 0.70 放寬量縮標準 → 調至 0.95
paramLookback 13 週 往前看幾根週K棒確認曾有高乖離（13週≒3個月） 抓更長週期高點 → 調至 20–26 週 只看近期 → 調至 8 週
paramVolMa 10 週 計算均量所用的週期數，作為量縮比較基準 一般不需調整，維持 10 週即可
paramPeakDelay 2 週 峰值需在幾根週K棒前以上，才確認回測已啟動（排除本週才見頂） 確認更充分的回測 → 調至 3 週 放寬以抓更早期機會 → 維持 2 週
paramLongBlack 3.0 % 長黑實體門檻：（開盤 − 收盤）/ 開盤 ≥ 此值，視為長黑K棒 更嚴格定義長黑 → 調至 4–5% 放寬偵測 → 調至 2%
paramHeavyVol 1.5 倍 爆量門檻：當週成交量 > 均量 × 此倍數，視為爆量 更嚴格定義爆量 → 調至 2.0 倍 放寬偵測 → 調至 1.3 倍

九宮格輸出欄位

OutputField 1
當前乖離率（%）
OutputField 2
近高回檔幅度（%）
OutputField 3
峰谷落差（%）
OutputField 4
峰值距今（週）
OutputField 5
谷底距今（週）
OutputField 6
量 / 均量比值
OutputField 7
期間最大正乖離（%）
OutputField 8
壓力量（週加總）
OutputField 9
支撐量（週加總）
OutputField 10
高檔長黑爆量（次）
壓力 / 支撐量判讀：OutputField 8（壓力）= 當前價位之上各週成交量加總；OutputField 9（支撐）= 前低至當前價位之間各週成交量加總。支撐量 > 壓力量，代表底部籌碼較紮實。

判讀重點

支
支撐量 > 壓力量
OutputField 9 > OutputField 8
底部籌碼紮實，上方解套賣壓較輕。支撐與壓力的量體差距越大，反彈動能越充裕。
幅
峰谷落差 5 – 20%
OutputField 3
此區間為正常洗盤幅度。落差 < 5% 可能是假回測；落差 > 25% 代表結構可能已遭破壞，需人工確認趨勢是否仍在。
底
谷底距今 0 – 1 週
OutputField 5
本週或上週剛見谷底，代表股價可能正在 週MA20 附近獲得支撐並啟動，是最積極的介入時機點。
檔
近高回檔幅度
OutputField 2
從近期最高週收盤回檔至今日的幅度。搭配「峰谷落差」可判斷目前是否已接近谷底，或仍在修正途中。
量
量 / 均量比值 < 0.85
OutputField 6
量縮代表回測期間無恐慌性賣壓，籌碼穩定換手。比值越低（如 0.5 以下），洗盤品質越好。
離
當前乖離率 ±3% 內
OutputField 1
乖離率接近 0 代表股價貼近週MA20。若為正乖離（0 ~ +3%），均線仍在下方支撐；若為負乖離（-3% ~ 0），股價暫跌破均線，需確認下週能否收復。
黑
高檔長黑爆量警訊
OutputField 10
專門檢查近期峰值那根週K棒是否同時出現長黑（實體跌幅 ≥ 3%）與爆量（> 均量 1.5 倍）。
0 → 峰值收盤正常，無出貨疑慮；1 → 高點那週長黑爆量，此波回測可能是主力出貨，建議放棄或等更多確認。

XScript 完整程式碼

週MA20量縮回測選股.xs 
// ============================================================
// 策略名稱：週MA20量縮回測選股
// 執行環境：選股腳本（Screener）/ 週K線圖
// 策略邏輯：
//   1. 週MA20 斜率向上（多頭趨勢確認）
//   2. 近期曾有顯著正乖離（股價一度強勢衝高）
//   3. 峰值已在 2 週前以上（確認回測已啟動）
//   4. 目前以量縮方式回測（修正非恐慌賣壓）
//   5. 當前乖離率已收斂至週MA20附近
// 版本日期：2026-05-08
// 修正紀錄：
//   v2 - 使用 GetField("Close","W") 明確取週資料
//   v3 - 改用 xf_GetValue("W",Wma20,1) 取前一週MA20，消除警告
//   v4 - 加入 NthHighestBar 確認峰值已在 2 週前以上
//   v5 - 加入 NthLowestBar 谷底位置、峰谷落差、壓力/支撐量計算
//   v6 - 加入高檔長黑爆量警訊計數
//   v7 - 改用週 High 取峰值、週 Low（2x 回看）取谷底
// ============================================================

// 確保足夠的歷史 K 棒長度
SetTotalBar(260);

// ── 參數設定 ────────────────────────────────────────────
var: paramDevHigh(8.0);     // 正乖離觸發門檻（%）
var: paramDevNear(3.0);     // 回測完成門檻（%）
var: paramVolRatio(0.85);   // 量縮比例閾值
var: paramLookback(13);     // 回看週數
var: paramVolMa(10);        // 均量計算期數
var: paramPeakDelay(2);     // 峰值需在幾週前以上才確認回測啟動
var: paramLongBlack(3.0);   // 長黑實體門檻（%）：開盤 vs 收盤跌幅超過此值視為長黑
var: paramHeavyVol(1.5);    // 爆量倍數門檻：成交量 > 均量 × 此倍數視為爆量

// ── 週資料（明確指定頻率）───────────────────────────────
var: wClose(0), wVol(0), Wma20(0);

wClose = GetField("Close", "W");
wVol   = GetField("Volume", "W");
Wma20  = Average(GetField("Close", "W"), 20);

// ── 計算核心變數 ─────────────────────────────────────────
var: devNow(0);
var: highestHigh(0), highestDev(0);
var: lowestLow(0), peakToTrough(0);
var: volMaVal(0), volRatioNow(0);
var: peakBar(0), troughBar(0);
var: pullbackPct(0);

// 當前日收盤對週MA20 的乖離率（%）
if Wma20 <> 0 then
    devNow = (GetField("Close", "D") - Wma20) / Wma20 * 100
else
    devNow = 0;

// 回看期間週高點最高值 / 週低點最低值（低點用 2x 回看抓更深支撐）
highestHigh = Highest(GetField("High", "W"), paramLookback);
lowestLow   = Lowest(GetField("Low", "W"), 2*paramLookback);

// 歷史最大正乖離率（%）
if Wma20 <> 0 then
    highestDev = (highestHigh - Wma20) / Wma20 * 100
else
    highestDev = 0;

// 峰谷落差（%）：從高點到回測谷底跌了多少（5-20% 為合理洗盤區間）
if highestHigh <> 0 then
    peakToTrough = (highestHigh - lowestLow) / highestHigh * 100
else
    peakToTrough = 0;

// 週均量與量縮比值
volMaVal = Average(wVol, paramVolMa);
if volMaVal <> 0 then
    volRatioNow = wVol / volMaVal
else
    volRatioNow = 1;

// 從近期最高週收盤回檔至今日收盤的幅度（%）
if highestHigh <> 0 then
    pullbackPct = (highestHigh - GetField("Close", "D")) / highestHigh * 100
else
    pullbackPct = 0;

// 峰值距今幾根週K棒（>=2 代表回測已啟動）
peakBar   = NthHighestBar(1, GetField("Close", "W"), paramLookback);

// 谷底距今幾根週K棒（0-1 代表剛找到支撐，可能正在啟動）
troughBar = NthLowestBar(1,  GetField("Close", "W"), paramLookback);

// ── 壓力量 / 支撐量計算 ──────────────────────────────────
var: i(0), volAbove(0), volBelow(0), curPrice(0);
var: alertCount(0), blackBody(0);   // 高檔長黑爆量警訊計數、實體幅度暫存

curPrice = GetField("Close", "D");

for i = 0 to paramLookback - 1 begin
    // 壓力：週收盤在當前價位之上
    if GetField("Close", "W")[i] > curPrice then
        volAbove = volAbove + GetField("Volume", "W")[i];

    // 支撐：週低點 >= 谷底且週收盤 <= 今日收盤
    if GetField("Low", "W")[i] >= lowestLow and
       GetField("Close", "W")[i] <= curPrice then
        volBelow = volBelow + GetField("Volume", "W")[i];

end;

// 高檔長黑爆量警訊：只檢查近期峰值那根週K棒
if GetField("Open", "W")[peakBar] <> 0 then
    blackBody = (GetField("Open", "W")[peakBar] - GetField("Close", "W")[peakBar])
                / GetField("Open", "W")[peakBar] * 100
else
    blackBody = 0;

if blackBody >= paramLongBlack
   and GetField("Volume", "W")[peakBar] > volMaVal * paramHeavyVol then
    alertCount = 1;

// ── 布林輔助旗標 ─────────────────────────────────────────
var: isMaRising(false);
var: hadHighDev(false);
var: isPeakPast(false);
var: isNearMa(false);
var: isVolShrink(false);

// 條件一：週MA20 上揚
if Wma20 > xf_GetValue("W", Wma20, 1) then
    isMaRising = true
else
    isMaRising = false;

// 條件二：曾有顯著正乖離
if highestDev >= paramDevHigh then
    hadHighDev = true
else
    hadHighDev = false;

// 條件三：峰值已在 paramPeakDelay 週前以上
if peakBar >= paramPeakDelay then
    isPeakPast = true
else
    isPeakPast = false;

// 條件四：當前乖離率收斂至均線附近
if devNow <= paramDevNear and devNow >= (-1 * paramDevNear) then
    isNearMa = true
else
    isNearMa = false;

// 條件五：量縮
if volRatioNow < paramVolRatio then
    isVolShrink = true
else
    isVolShrink = false;

// ── 選股觸發 ─────────────────────────────────────────────
if isMaRising = true
    and hadHighDev = true
    and isPeakPast = true
    and isNearMa = true
    and isVolShrink = true
then begin
    ret = 1;

    // 九宮格輸出欄位
    OutputField1(devNow,        "當前乖離率(%)");
    OutputField2(pullbackPct,   "近高回檔幅度(%)");
    OutputField3(peakToTrough,  "峰谷落差(%)");
    OutputField4(peakBar,       "峰值距今(週)");
    OutputField5(troughBar,     "谷底距今(週)");
    OutputField6(volRatioNow,   "量/均量比值");
    OutputField7(highestDev,    "期間最大正乖離(%)");
    OutputField8(volAbove,      "壓力量(週加總)");
    OutputField9(volBelow,      "支撐量(週加總)");
    OutputField10(alertCount,   "高檔長黑爆量(次)");
end;
